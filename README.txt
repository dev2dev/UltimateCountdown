ULTIMATE COUNTDOWN

Ultimate Countdown is an electronic "chicken" game for two players for the 
iPhone. Each player holds down their big red button while the timer quickly 
counts down to zero. The last player to let go is the winner -- but only if 
the timer hasn't hit zero yet!
